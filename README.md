# SimpleLedger

##  概要

SimpleLedgerは、スマートフォンで手軽に使える家計簿アプリです。
電卓のような直感的な操作で支出を入力し、日別・月別の集計を自動で確認できます。

本アプリは、日常的に実際に利用することを目的として制作しました。

---

##  アクセス

https://simpleledger.onrender.com

---

##  主な機能

*  電卓UIによる支出入力（Ajaxで即時保存）
*  日別 / 月別の支出集計
*  カテゴリごとの管理
*  ユーザーごとのデータ分離（ログイン機能）
*  スマホ最適化UI（PWA対応）

##  使用技術

* Python
* Django
* PostgreSQL
* Render（デプロイ環境）
* HTML / CSS / JavaScript
* WhiteNoise（静的ファイル配信）

---

##  使い方

1. アクセスURLまたはQRコードからアプリを開く
2. 新規登録 / ログイン
3. 電卓UIで金額を入力
4. カテゴリを選択して保存
5. 集計画面で支出を確認
